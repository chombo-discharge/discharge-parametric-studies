Get the source
==============

Clone the repository from GitHub.  The clone contains the Python package
(``discharge_inception/``), the SLURM shell wrapper
(``Util/GenericArrayJob.sh``), example jobscripts, the ``Exec/Rod`` case, and
this documentation source.  No submodules are used, so a plain ``git clone``
is sufficient.

.. code-block:: bash

   git clone https://github.com/SINTEF-Power-system-asset-management/discharge-inception.git
   cd discharge-inception
